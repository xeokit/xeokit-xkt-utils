    public loadGLTFIntoXKTModel(gltf: Object, model: XKTModel, options: Object): Promise source 

    import {loadGLTFIntoXKTModel} from 'xeokit-xkt-tools/src/XKTModel/loadGLTFIntoXKTModel.js'

    Parses glTF JSON into an XKTModel.

    Expects the XKTModel to be freshly instantiated, and calls XKTModel#finalize on the XKTModel before returning.

    Params:

    NameTypeAttributeDescription
    gltf Object

    The glTF JSON.
    model XKTModel

    XKTModel to parse into
    options Object
    options.basePath String
    • optional

    Base directory where binary attachments may be found.

    Return:

    Promise

    A Promise which returns the XKTModel when resolved.
